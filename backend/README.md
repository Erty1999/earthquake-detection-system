### Configuration
Before running the software, add these variables to the .env file:

#Port
PORT=

#first Admin account
ADMIN_FN=
ADMIN_LN=
ADMIN_BIRTHDAY=
ADMIN_EMAIL=
ADMIN_PWD=

#etl account
ETL_FN=
ETL_LN=
ETL_BIRTHDAY=
ETL_EMAIL=
ETL_PWD=

#passive devices manager account
PDM_FN=
PDM_LN=
PDM_BIRTHDAY=
PDM_EMAIL=
PDM_PWD=

#ETL base URL
ETL_BASE_URL=

#PDM base URL
ETL_BASE_URL=

### `yarn install`
Install all required packages

### `yarn dev`
Run the server in development mode.
