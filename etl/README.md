# Configuration
Before running the software, add these variables to the .env file:

```bash
## Port 
PORT=

# ETL credentials 
ETL_EMAIL=
ETL_PWD=

## Backend base URL
BE_BASE_URL=
```
<br>

# Local
To start this service locally use the following commands:

```bash
#Install all required packages
yarn install

#Run the server in development mode.
yarn dev
```
